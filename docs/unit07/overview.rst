Unit 7: Jupyter and Visualization for Data Analysis
===================================================
In this unit, we will learn about Jupyter Notebooks and how to use them for interactive
computing and data visualization. We will learn how to create and run Jupyter Notebooks
and how to use them to document our code and share our work with others. In the process of
working with them, we will explore some popular libraries, including Matplotlib, a powerful
plotting library for Python; Seaborn, a library for statistical data visualization built on
top of Matplotlib; and pandas, a library for data manipulation and analysis, to do some
exploratory data analysis and visualization.
