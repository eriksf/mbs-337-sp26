Unit 9: Continuous Integration / Continuous Deployment
======================================================

Up to this point, we have covered many topics on the *DevOps* (developement and operations) of research
computing. This expecially includes **Git** for version control, **Docker** for containerization, and 
**Compose** for container orchestration. These topics go beyond simply writing or running code for 
computational biology. Rather they focus on the process and the environment so that code is written 
or run in a reliable and reproducible way.

This unit explores additional topics that fall under the DevOps of research computing. The topics 
focus on automating the build, testing, and deployment process of your software so that you can work
on projects, including collaboratively, quickly and efficiently.
